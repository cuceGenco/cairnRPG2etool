Character Creation

Attribute generation: 3d6 STR/DEX/WIL (swap two optional)
HP: 1d6 base
Backgrounds (example): Acolyte, Artisan, Criminal, Entertainer, Folk Hero, etc. Set skills, equipment, story hook.
Bonds/Omen: roll random bond and omen for narrative push.

Combat / Initiative

Initiative is run with DEX. DM uses round order from initiative tracker.
Combat modifiers: impaired/enhanced, enemy morale check +2 etc. (Cairn book).

Wilderness / Encounters

Use topography table, points-of-interest + event table. Random encounter: 1d6 choices.

Bestiary

Undead: Wendigo. Beast: Grim Wolf. Humanoid: Cairn Raider. Abomination: Stonecrawler. Build using Challenge, HP, Attack, Traits.
